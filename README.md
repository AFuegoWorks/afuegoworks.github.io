# AFuegoWorks landing page

This version is a self-contained `index.html`. The AF logo is embedded directly into the file, so you do not need to upload an assets folder.

## Publish

1. Replace the existing `index.html` in your GitHub Pages repository.
2. Keep the existing `CNAME` file.
3. Commit and push to `main`.
4. Wait for GitHub Pages to redeploy.

## Not included

- The Discord URL provided was a private `/channels/@me` link, not a public contact link. Add a public server invite later if you create one.
- The Fourthwall URL was an admin dashboard link, not a public storefront.
- Replit was a generic account URL rather than a public profile.
- Patreon, Ko-fi, Buy Me a Coffee, Facebook, Pinterest, Threads, and Reddit were left off to keep the routing page focused.
